SCORM Vocabulary

Editorial Note:

The SCORM profile includes Verbs, Activity Types and xAPI Document definitions used to represent SCORM learning experiences in xAPI. Note: Vocabulary terms that are not managed by the SCORM profile and referenced from other controlled vocabularies are higlighted in yellow .

Published by: SCORM Profile Working Group
Date Created: 2016-6-09
Last Modified: 2016-7-11

Verbs

Label Scope Note ID (IRI) Vocabulary
commented Used to indicate comments from the learner during the current activity attempt http://adlnet.gov/expapi/verbs/commented https://w3id.org/xapi/adl
completed Used to indicate the learner has experienced a sufficient amount of the activity on this attempt. https://w3id.org/xapi/adl/verbs/completed https://w3id.org/xapi/adl
failed Used to indicate that the learner's performance on this attempt did not meet the minimum threshold for the activity. http://adlnet.gov/expapi/verbs/failed https://w3id.org/xapi/adl
initialized Used to indicate that the SCO attempt has been initialized. http://adlnet.gov/expapi/verbs/initialized https://w3id.org/xapi/adl
passed Used to indicate that the learner's performance on this attempt at least met the minimum threshold for the activity. http://adlnet.gov/expapi/verbs/passed https://w3id.org/xapi/adl
progressed Used to indicate the learner's progress measure within an activity attempt. http://adlnet.gov/expapi/verbs/progressed https://w3id.org/xapi/adl
responded Used to indicate the learner's response to some interaction with this activity on this attempt. http://adlnet.gov/expapi/verbs/responded https://w3id.org/xapi/adl
resumed Used to indicate that the SCO attempt has been resumed. http://adlnet.gov/expapi/verbs/resumed https://w3id.org/xapi/adl
scored Used to indicate the learner's score on this attempt without implying a level of satisfaction, or to provide evidence toward satisfaction or completion. http://adlnet.gov/expapi/verbs/scored https://w3id.org/xapi/adl
suspended Used to indicate that the SCO attempt has been suspended. http://adlnet.gov/expapi/verbs/suspended https://w3id.org/xapi/adl
terminated Used to indicate that the SCO attempt has been terminated. http://adlnet.gov/expapi/verbs/terminated https://w3id.org/xapi/adl
