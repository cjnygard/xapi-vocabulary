动词 (Verbs)

动词 描述 用法说明 ID (IRI) Related Match Closely Related Term 限制词
回答 指 actor 回答一个问题，其 object 通常用 activity 来表示该问题，回答的内容通常会列在 result 里的 response 处. 指 actor 回答某一测评题或习题的答案. http://adlnet.gov/expapi/verbs/answered https://w3id.org/xapi/adl
提问 指 actor 提出询问期望获得回应或是问题的答案. 指 actor 在论坛或其他 web 应用程序提出一个问题. http://adlnet.gov/expapi/verbs/asked https://w3id.org/xapi/adl
开始尝试 指 actor 努力取得或使用该 object。 在某些情况下，一个开始尝试的陈述 (statement) 如果没有额外的 activity，可能会被视为未完成. 指 actor 开始尝试一个评量. http://adlnet.gov/expapi/verbs/attempted https://w3id.org/xapi/adl
加书籤 指读者在其停止电子书 activity 或是认为重要的位置 (页面) 做标记. https://w3id.org/xapi/adb/verbs/bookmarked https://w3id.org/xapi/adb
回应 指 actor 对某一 object 提供手写或数字注释. 指在一个论坛或其他 web 应用程序中添加注释. http://adlnet.gov/expapi/verbs/commented https://w3id.org/xapi/adl
完成 指 actor 正常地完成或结束活动. http://adlnet.gov/expapi/verbs/completed https://w3id.org/xapi/adl
设计 指 actor 为特定目的仔细计划或创建該 object. 指教师设计一套课程计划/学习计划/行动计划. https://w3id.org/xapi/acrossx/verbs/designed http://wordnet-rdf.princeton.edu/wn31/201757273-v https://w3id.org/xapi/acrossx
不贊同 指 actor 不赞同该 object 或 activity. 指在论坛或社交媒体对某贴文或响应表示否定. https://w3id.org/xapi/acrossx/verbs/disliked http://wordnet-rdf.princeton.edu/wn31/201757273-v https://w3id.org/xapi/acrossx
编辑 指 actor 在档案或 object 添加或修改内容. 指在电子表格或论坛贴文中增加或修改内容. https://w3id.org/xapi/acrossx/verbs/edited http://wordnet-rdf.princeton.edu/wn31/200629787-v https://w3id.org/xapi/acrossx
评分/评价 指 actor 形成意见或准备评量另一个 actor 或 object. 指一位老师/教练评价学习者的表现或作品. https://w3id.org/xapi/acrossx/verbs/evaluated http://wordnet-rdf.princeton.edu/wn31/200672179-v https://w3id.org/xapi/acrossx
画重点 指为往后复习或做笔记之目的,在内容里标示重点区域. https://w3id.org/xapi/adb/verbs/highlighted https://w3id.org/xapi/adb
互动 指 actor 与一个实体或虚拟的 object 互动. 指一个学习者与内容中的互动元素进行互动的行为.(例如测评题的答案选项). http://adlnet.gov/expapi/verbs/interacted https://w3id.org/xapi/adl
按赞 指 actor 在一个论坛或社群媒体应用程序赞同一篇贴文或是一个人. 指在论坛或社交媒体对某贴文或响应表示赞同. https://w3id.org/xapi/acrossx/verbs/liked closeMatch: http://activitystrea.ms/schema/1.0/like https://w3id.org/xapi/acrossx
登录 指 actor 经由识别和认证由 actor 所提供的凭证获得存取到系统或服务的权限. https://w3id.org/xapi/adl/verbs/logged-in https://w3id.org/xapi/adl
做笔记 指将注释或笔记添加到电子书内所选本文. https://w3id.org/xapi/adb/verbs/noted https://w3id.org/xapi/adb
张贴 指在论坛或社群媒体张贴一个讨论主题或讯息. https://w3id.org/xapi/acrossx/verbs/posted http://wordnet-rdf.princeton.edu/wn31/200993715-v https://w3id.org/xapi/acrossx
阅读 指 actor 读取此 object。 通常仅适用于对象代表印刷或书写的内容，如一本书、一条讯息或是一个评论. https://w3id.org/xapi/adb/verbs/read https://w3id.org/xapi/adb
查询参考 指使用辞典或其他资源来查询所选字汇. 一个学习者可以参考外部的信息来支持某主题, 也可以参考其他阅读材料给他/她的同事或导师. 指 actor 透过一个链接查询某一个在线资源. https://w3id.org/xapi/adb/verbs/referenced https://w3id.org/xapi/adb
报告 用来描述 actor 自我报告一项学习经验的行动, 但也可以用于报告其他人的行动. 指 actor 发布某项经验或活动的结果. https://w3id.org/xapi/acrossx/verbs/reported http://wordnet-rdf.princeton.edu/wn31/200968841-v https://w3id.org/xapi/acrossx
求助 指 actor 需要或请求一个 object 或是另一位 actor. 指 actor 请求同侪或教练/教师的协助. https://w3id.org/xapi/adb/verbs/requested https://w3id.org/xapi/adb
搜寻 指 actor 在某 object 寻找信息. 指 actor 以一串关键词在某应用程序或接口搜索信息. https://w3id.org/xapi/acrossx/verbs/searched closeMatch: http://activitystrea.ms/schema/1.0/search http://wordnet-rdf.princeton.edu/wn31/201318273-v https://w3id.org/xapi/acrossx
分享 指 actor 向其他 actors 或群体公开提供共同感兴趣 object 的访问权限的意图. 指 actor 分享内容或额外的参考材料到另一个应用程序或是用户. http://adlnet.gov/expapi/verbs/shared https://w3id.org/xapi/adl
跳过 指 actor 在时间轴或或预設架构序列上非连续性的使用内容, 例如，视频、音频文件或电子书. 指 actor 观看影片时向前或向后略过. http://id.tincanapi.com/verb/skipped https://registry.experienceapi.com
揭示 指 actor 公开原本被隐藏的内容或 object. 指 actor 开启ㄧ个信息或考题窗口来读取或回答它. https://w3id.org/xapi/acrossx/verbs/revealed https://w3id.org/xapi/acrossx
被指定 指 actor 被给予或要求存取一个 object 或是完成一项任务. 指一个学习者被分配某学习材料的动作. https://w3id.org/xapi/acrossx/verbs/was-assigned http://wordnet-rdf.princeton.edu/wn31/202396945-v https://w3id.org/xapi/acrossx
观看 指 actor 观看一个视觉的 object. 指 actor 观看一段视频. https://w3id.org/xapi/adb/verbs/watched https://w3id.org/xapi/adb

活动类型 (Activity Types)

Acitivity Type Description Scope Note ID (IRI) Related Match Closely Related Term Vocabulary
测评 测评是一种 activity 型式, 用以确认学习者熟练某个学科领域. 测评通常包含一个或多个测评题. 指以数字形式进行的评量测验，用以区别纸张形式的评量测验. http://adlnet.gov/expapi/activities/assessment https://w3id.org/xapi/adl
协作 一个 activity 有多位 actors 为一项作业、项目、问题问题的挑战或目标而协同合作. 指共同合作解决问题的 activity. https://w3id.org/xapi/acrossx/activities/collaboration http://wordnet-rdf.princeton.edu/wn31/101207646-n https://w3id.org/xapi/acrossx
电书/简报/课件 一本以電子形式呈現文件、檔案或是互動式應用程式的書, 具有可選擇格式設置和控制功能. https://w3id.org/xapi/acrossx/activities/e-book http://wordnet-rdf.princeton.edu/wn31/106522968-n https://w3id.org/xapi/acrossx
当面讨论 两个或以上的 actors 在一真实地点亲自面对面针对某主题交换意见或沟通. https://w3id.org/xapi/acrossx/activities/face-to-face-discussion http://wordnet-rdf.princeton.edu/wn31/107155367-n https://w3id.org/xapi/acrossx
CMI 测评题 通常是一个大 activity (如评量、游戏或仿真) 的一部分，并且具有让学习者提供输入的控制. 它可以是独立的 asset 或 function. http://adlnet.gov/expapi/activities/interaction https://w3id.org/xapi/adl
课堂即时互动系统 一种 activity 可以让教师创建内建测验/投票问题的投影片, 用以收集学习者响应，并能在班级内立即产生报告. https://w3id.org/xapi/acrossx/activities/instant-response-system https://w3id.org/xapi/acrossx
学习计画 一个学习计划是用来计划或组织不同来源的学习活动，它可以由老师、学习设计师或学习者本身建立. https://w3id.org/xapi/acrossx/activities/learning-plan http://wordnet-rdf.princeton.edu/wn31/105736623-n https://w3id.org/xapi/acrossx
简讯 个别或群体 actors 之间传达的信息. Actors 之间的个人讯息, 通常是非公开的传递. https://w3id.org/xapi/acrossx/activities/message http://wordnet-rdf.princeton.edu/wn31/100033319-n https://w3id.org/xapi/acrossx
笔记 学习者在任何学习内容或活动中所做的注释或笔记. https://w3id.org/xapi/acrossx/activities/note http://wordnet-rdf.princeton.edu/wn31/106775984-n https://w3id.org/xapi/acrossx
线上论坛 二个或以上之 actors 在在线针对某主题进行意见交流或沟通. https://w3id.org/xapi/acrossx/activities/online-discussion http://wordnet-rdf.princeton.edu/wn31/107155367-n https://w3id.org/xapi/acrossx
页 指一本电子或实体书面材料的页码. 指一本电子书或印刷书籍的页码. https://w3id.org/xapi/acrossx/activities/page http://wordnet-rdf.princeton.edu/wn31/106269333-n https://w3id.org/xapi/acrossx
纸书 书面的著作或作品被实际打印到纸张上并装订在一起. https://w3id.org/xapi/acrossx/activities/printed-book http://wordnet-rdf.princeton.edu/wn31/106422547-n https://w3id.org/xapi/acrossx
学习单 一页或数页面包含问题、练习或学习活动指示. https://w3id.org/xapi/acrossx/activities/printed-worksheet http://wordnet-rdf.princeton.edu/wn31/106427434-n https://w3id.org/xapi/acrossx
纸笔测验 书面的考试或测试，以纸张形式提供并用笔填写. https://w3id.org/xapi/acrossx/activities/printed-assessment http://wordnet-rdf.princeton.edu/wn31/107213988-n https://w3id.org/xapi/acrossx
问题 问题通常是评量的一部分，需要从学习者获得回应然后评估其正确性. http://adlnet.gov/expapi/activities/question https://w3id.org/xapi/adl
视频 一段具有声音和视觉的录制内容，并可在视频片播放器上显示. 视频是具有音频和视觉组件的内容，可以提供联机或脱机观看. https://w3id.org/xapi/acrossx/activities/video closeMatch: http://activitystrea.ms/schema/1.0/video http://wordnet-rdf.princeton.edu/wn31/104541422-n https://w3id.org/xapi/acrossx
网页 网络页面文档 (通常是 HTML 格式)，任何人都可透过连接到互联网和使用网络浏览器来存取. 任何具有 IRL 或 URL 的网页. https://w3id.org/xapi/acrossx/activities/webpage closeMatch: http://activitystrea.ms/schema/1.0/page http://wordnet-rdf.princeton.edu/wn31/106370307-n https://w3id.org/xapi/acrossx
