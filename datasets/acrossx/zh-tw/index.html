動詞 (Verbs)

動詞 描述 用法說明 ID (IRI) Related Match Closely Related Term 限制詞
回答 指 actor 回答一個問題，其 object 通常用 activity 來表示該問題，回答的內容通常會列在 result 裡的 response 處。 指 actor 回答某ㄧ考題或習題的答案。 http://adlnet.gov/expapi/verbs/answered https://w3id.org/xapi/adl
提問 指 actor 提出詢問期望獲得回應或是問題的答案. 指 actor 在論壇或其他 web 應用程式提出一個問題. http://adlnet.gov/expapi/verbs/asked https://w3id.org/xapi/adl
開始嘗試 指 actor 努力取得或使用該 object。在某些情況下，一個開始嘗試的陳述 (statement) 如果沒有額外的 activity，可能會被視為未完成. 指 actor 開始嘗試一個評量. http://adlnet.gov/expapi/verbs/attempted https://w3id.org/xapi/adl
加書籤 指讀者在其停止電子書 activity 或是認為重要的位置 (頁面) 做標記. https://w3id.org/xapi/adb/verbs/bookmarked https://w3id.org/xapi/adb
回應 指 actor 對某ㄧ object 提供數位或手寫注釋. 指在一個論壇或其他 web 應用程式中添加注釋. http://adlnet.gov/expapi/verbs/commented https://w3id.org/xapi/adl
完成 指 actor 正常地完成或結束活動. http://adlnet.gov/expapi/verbs/completed https://w3id.org/xapi/adl
設計 指 actor 為特定目的仔細計畫或創建該 object. 指教師設計一套課程計劃/學習計劃/行動計畫. https://w3id.org/xapi/acrossx/verbs/designed http://wordnet-rdf.princeton.edu/wn31/201757273-v https://w3id.org/xapi/acrossx
不贊同 指 actor 不贊同該 object 或 activity. 指在論壇或社交媒體對某貼文或回應表示否定. https://w3id.org/xapi/acrossx/verbs/disliked http://wordnet-rdf.princeton.edu/wn31/201757273-v https://w3id.org/xapi/acrossx
編輯 指 actor 在檔案或 object 添加或修改內容. 指在試算表或論壇貼文中增加或修改內容. https://w3id.org/xapi/acrossx/verbs/edited http://wordnet-rdf.princeton.edu/wn31/200629787-v https://w3id.org/xapi/acrossx
評分/評價 指 actor 形成意見或準備評量另一個 actor 或 object. 指一位老師/教練評價學習者的表現或作品. https://w3id.org/xapi/acrossx/verbs/evaluated http://wordnet-rdf.princeton.edu/wn31/200672179-v https://w3id.org/xapi/acrossx
畫重點 指為往後複習或做筆記之目的,在內容裡標示重點區域. https://w3id.org/xapi/adb/verbs/highlighted https://w3id.org/xapi/adb
互動 指 actor 與一個實體或虛擬的 object 互動. 指一個學習者與內容中的互動元素進行互動的行為.(例如測驗題的答案選項) http://adlnet.gov/expapi/verbs/interacted https://w3id.org/xapi/adl
按讚 指 actor 在一個論壇或社群媒體應用程式贊同ㄧ篇貼文或是一個人. 指在論壇或社群媒體對某貼文或回應表示贊同. https://w3id.org/xapi/acrossx/verbs/liked closeMatch: http://activitystrea.ms/schema/1.0/like https://w3id.org/xapi/acrossx
登入 指 actor 經由識別和認證由 actor 所提供的憑證獲得存取到系統或服務的權限. https://w3id.org/xapi/adl/verbs/logged-in https://w3id.org/xapi/adl
做筆記 指將注釋或筆記添加到電子書內所選本文. https://w3id.org/xapi/adb/verbs/noted https://w3id.org/xapi/adb
張貼 指在論壇或社群媒體張貼一個討論主題或訊息. https://w3id.org/xapi/acrossx/verbs/posted http://wordnet-rdf.princeton.edu/wn31/200993715-v https://w3id.org/xapi/acrossx
閱讀 指 actor 讀取此 object. 通常僅適用於物件代表印刷或書寫的內容，如一本書、一條訊息或是一個評論. https://w3id.org/xapi/adb/verbs/read https://w3id.org/xapi/adb
查詢參考 指使用辭典或其他資源來查詢所選字彙. 一個學習者可以參考外部的資訊來支援某主題, 也可以參考其他閱讀材料給他/她的同事或導師. 指 actor 透過ㄧ個連結查詢某ㄧ個線上資源. https://w3id.org/xapi/adb/verbs/referenced https://w3id.org/xapi/adb
報告 用來描述 actor 自我報告ㄧ項學習經驗的行動, 但也可以用於報告其他人的行動. 指 actor 發布某項經驗或活動的結果. https://w3id.org/xapi/acrossx/verbs/reported http://wordnet-rdf.princeton.edu/wn31/200968841-v https://w3id.org/xapi/acrossx
求助 指 actor 需要或請求一個 object 或是另一位 actor. 指 actor 請求同儕或教練/教師的協助. https://w3id.org/xapi/adb/verbs/requested https://w3id.org/xapi/adb
搜尋 指 actor 在某 object 尋找資訊. 指 actor 以一串關鍵字在某應用程式或介面搜索資訊. https://w3id.org/xapi/acrossx/verbs/searched closeMatch: http://activitystrea.ms/schema/1.0/search http://wordnet-rdf.princeton.edu/wn31/201318273-v https://w3id.org/xapi/acrossx
分享 指 actor 向其他 actors 或群體公開提供共同感興趣 object 的存取權限. 指 actor 分享內容或額外的參考材料到另一個應用程式或是使用者. http://adlnet.gov/expapi/verbs/shared https://w3id.org/xapi/adl
跳過 指 actor 在時間軸或預設架構序列上非連續性的使用內容, 例如，影片、音訊檔或電子書. 指 actor 觀看影片時向前或向後略過. http://id.tincanapi.com/verb/skipped https://registry.experienceapi.com
揭示 指 actor 公開原本被隱藏的內容或 object. 指 actor 開啟ㄧ個資訊或考題視窗來讀取或回答它. https://w3id.org/xapi/acrossx/verbs/revealed https://w3id.org/xapi/acrossx
被指定 指 actor 被給予或要求存取ㄧ個 object 或是完成一項任務. 指一個學習者被分配某學習材料的動作. https://w3id.org/xapi/acrossx/verbs/was-assigned http://wordnet-rdf.princeton.edu/wn31/202396945-v https://w3id.org/xapi/acrossx
觀看 指 actor 觀看一個視覺的 object. 指 actor 觀看一段影片. https://w3id.org/xapi/adb/verbs/watched https://w3id.org/xapi/adb

活動類型 (Activity Types)

Acitivity Type Description Scope Note ID (IRI) Related Match Closely Related Term Vocabulary
評量測驗 評量是一種 activity 型式, 用以確認學習者熟練某個學科領域. 評量通常包含一個或多個試題. 指以數位形式進行的評量測驗，用以區別紙張形式的評量測驗. http://adlnet.gov/expapi/activities/assessment https://w3id.org/xapi/adl
協作 一個 activity 有多位 actors 為一項作業、專案、問題問題的挑戰或目標而協同合作. 指共同合作解決問題的 activity. https://w3id.org/xapi/acrossx/activities/collaboration http://wordnet-rdf.princeton.edu/wn31/101207646-n https://w3id.org/xapi/acrossx
電子書/簡報/課件 ㄧ本以電子形式呈現文件、檔案或是互動式應用程式的書, 具有可選擇格式設置和控制功能. https://w3id.org/xapi/acrossx/activities/e-book http://wordnet-rdf.princeton.edu/wn31/106522968-n https://w3id.org/xapi/acrossx
當面討論 兩個或以上的 actors 在一真實地點親自面對面針對某主題交換意見或溝通. https://w3id.org/xapi/acrossx/activities/face-to-face-discussion http://wordnet-rdf.princeton.edu/wn31/107155367-n https://w3id.org/xapi/acrossx
CMI 題目 通常是一個大 activity (如評量、遊戲或模擬) 的一部分，並且具有讓學習者提供輸入的控制. 它可以是獨立的 asset 或 function. http://adlnet.gov/expapi/activities/interaction https://w3id.org/xapi/adl
課堂即時互動系統 一種 activity 可以讓教師創建內建測驗/投票問題的投影片, 用以收集學習者回應，並能在班級內立即產生報告. https://w3id.org/xapi/acrossx/activities/instant-response-system https://w3id.org/xapi/acrossx
學習計畫 一個學習計畫是用來計畫或組織不同來源的學習活動，它可以由老師、學習設計師或學習者本身建立. https://w3id.org/xapi/acrossx/activities/learning-plan http://wordnet-rdf.princeton.edu/wn31/105736623-n https://w3id.org/xapi/acrossx
簡訊 個別或群體 actors 之間傳達的資訊. Actors 之間的個人訊息, 通常是非公開的傳遞. https://w3id.org/xapi/acrossx/activities/message http://wordnet-rdf.princeton.edu/wn31/100033319-n https://w3id.org/xapi/acrossx
筆記 學習者在任何學習內容或活動中所做的注釋或筆記. https://w3id.org/xapi/acrossx/activities/note http://wordnet-rdf.princeton.edu/wn31/106775984-n https://w3id.org/xapi/acrossx
線上討論 二個或以上之 actors 在線上針對某主題進行意見交流或溝通. https://w3id.org/xapi/acrossx/activities/online-discussion http://wordnet-rdf.princeton.edu/wn31/107155367-n https://w3id.org/xapi/acrossx
頁 指一本電子或實體書面材料的頁碼. 指一本電子書或印刷書籍的頁碼. https://w3id.org/xapi/acrossx/activities/page http://wordnet-rdf.princeton.edu/wn31/106269333-n https://w3id.org/xapi/acrossx
印刷書 書面的著作或作品被實際列印到紙張上並裝訂在一起. https://w3id.org/xapi/acrossx/activities/printed-book http://wordnet-rdf.princeton.edu/wn31/106422547-n https://w3id.org/xapi/acrossx
學習單 ㄧ頁或數頁面包含問題、練習或學習活動指示. https://w3id.org/xapi/acrossx/activities/printed-worksheet http://wordnet-rdf.princeton.edu/wn31/106427434-n https://w3id.org/xapi/acrossx
紙筆測驗 書面的考試或測試，以紙張形式提供並用筆填寫. https://w3id.org/xapi/acrossx/activities/printed-assessment http://wordnet-rdf.princeton.edu/wn31/107213988-n https://w3id.org/xapi/acrossx
問題 問題通常是評量的一部分，需要從學習者獲得回應然後評估其正確性. http://adlnet.gov/expapi/activities/question https://w3id.org/xapi/adl
影片 一段具有聲音和視覺的錄製內容，並可在影片播放器上顯示. 影片是具有音訊和視覺元件的內容，可以提供連線或離線觀看. https://w3id.org/xapi/acrossx/activities/video closeMatch: http://activitystrea.ms/schema/1.0/video http://wordnet-rdf.princeton.edu/wn31/104541422-n https://w3id.org/xapi/acrossx
網頁 網路頁面檔案 (通常是 HTML 格式)，任何人都可透過連接到互聯網和使用網路瀏覽器來存取. 任何具有 IRL 或 URL 的網頁. https://w3id.org/xapi/acrossx/activities/webpage closeMatch: http://activitystrea.ms/schema/1.0/page http://wordnet-rdf.princeton.edu/wn31/106370307-n https://w3id.org/xapi/acrossx
