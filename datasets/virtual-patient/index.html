Verbs

Label Scope Note ID (IRI) Vocabulary Reference
arrived User has visited a node within the VP case. Not necessarily same as 'read' or ‘experienced’ because an internal rule or trigger might immediately bounce them on to another node or activity. (See ‘Launched’). So, while the user may have landed on that node, if they are immediately redirected, they may not even see that page/node. Can be used for a physical or virtual location. https://w3id.org/xapi/adb/verbs/arrived https://w3id.org/xapi/adb
commented User has contributed to group activity. Used when an actor adds to a group activity for a compound effect. In some ways, this is not different from ‘responded’. But the net effect is different because you are not expecting that the single activity will necessarily produce some result by itself. The object might be a discussion thread. (Or in OpenLabyrinth, we have a question format called a Cumulative Question, where users contribute phrases to a single question/database field.) http://adlnet.gov/expapi/verbs/commented https://w3id.org/xapi/adl
completed User has completed the case. Reached a node designated by the case author as an endpoint. Does not indicate success or failure. http://adlnet.gov/expapi/verbs/completed https://w3id.org/xapi/adl
experienced User has read a document, page or paragraph. Similar to ‘interacted’ but used when the media is static or non-interactive http://adlnet.gov/expapi/verbs/experienced https://w3id.org/xapi/adl
ignored https://w3id.org/xapi/medbiq/verbs/ignored https://w3id.org/xapi/medbiq
initialized Includes a timestamp and identifier of the node in the VP when playing was started. Most VP cases will have a fixed starting node but this is not always true and is not a requirement for the statement to be true. http://adlnet.gov/expapi/verbs/initialized https://w3id.org/xapi/adl
interacted User has interacted with or manipulated an object or a video or similar multimedia that provides some sort of interactivity. Similar to 'experienced' for when the media is dynamic, like a video. http://adlnet.gov/expapi/verbs/interacted https://w3id.org/xapi/adl
launched VP player engine has initiated an action such as redirecting the user to a new node, or triggered some other activity. This is not an action made by the user. http://adlnet.gov/expapi/verbs/launched https://w3id.org/xapi/adl
responded User has responded to a question. Response to an item or question within a node, without moving off that node. (This is not quite the same as ‘answered’ which implies that the answer is correct.) http://adlnet.gov/expapi/verbs/responded https://w3id.org/xapi/adl
resumed User has restarted a case. Resumed a case that had been previously suspended. http://adlnet.gov/expapi/verbs/resumed https://w3id.org/xapi/adl
suspended User has paused a case. Suspended a case with the possibility to resume later. http://adlnet.gov/expapi/verbs/supspended https://w3id.org/xapi/adl
terminated User has left a case and does not intend to resume. Consider how you will be able to differentiate this from ‘abandoned’, 'suspended' or 'completed' because the VP player engine may not be able to determine whether the user is coming back again. http://adlnet.gov/expapi/verbs/terminated https://w3id.org/xapi/adl
updated https://w3id.org/xapi/medbiq/verbs/updated https://w3id.org/xapi/medbiq
