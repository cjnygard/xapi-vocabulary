cmi5 Vocabulary

The cmi5 profile is a specification that expands xAPI into the LMS-centric, single-learner distributed learning model. It can be used in other ways, but the primary use case is to behave in a traditional LMS model. Note: Vocabulary terms that are not managed by the cmi5 project and referenced from other controlled vocabularies are higlighted in yellow .

Activity Types

Label Description Scope Note ID (IRI) Closely Related Term Vocabulary
block A block represents a number of Assignable Units of which progress (completion/success) is rolled up to. In cmi5 it is every level above the AU and below the Course. https://w3id.org/xapi/cmi5/activitytype/block https://w3id.org/xapi/cmi5
course A course represents an amount of content that is published and registered for with the purpose of gaining completion. It is represented with a Course Structure Format in cmi5 as the highest level of content (above Block and AU). https://w3id.org/xapi/cmi5/activitytype/course http://wordnet-rdf.princeton.edu/wn31/100886144-n https://w3id.org/xapi/cmi5

Activity Description Scope Note ID (IRI) Vocabulary

Result Extensions

Label Description Scope Note ID (IRI) Closely Related Term Vocabulary
progress An integer value bewteen 0 and 100 (inclusive) indicating the completion of the Activity/AU as a percentage. Used as a result extension. https://w3id.org/xapi/cmi5/result/extensions/progress https://w3id.org/xapi/cmi5
reason Indicates why the Activity/AU was waived. Used as a result extension. https://w3id.org/xapi/cmi5/result/extensions/reason https://w3id.org/xapi/cmi5
session Used in contexttemplate. Value is used to populate/update session_id. https://w3id.org/xapi/cmi5/extensions/session https://w3id.org/xapi/cmi5

Context Extensions

Extension Description Scope Note ID (IRI) Closely Related Term Vocabulary
sessionid A unique String identifier for a single AU launch session based on actor and course registration. Generated by LRS and must be in all cmi5 defined and cmi5 allowed Statements. Used as a context extension. https://w3id.org/xapi/cmi5/context/extensions/sessionid https://w3id.org/xapi/cmi5
masteryscore Decimal value between 0 and 1 indicating the score needed to provide a passed Statement. Determined on an AU plus registration basis. Used as a context extension. https://w3id.org/xapi/cmi5/context/extensions/masteryscore https://w3id.org/xapi/cmi5
launchmode String indicating which mode the AU was launched with by the LMS. Used as a context extension. https://w3id.org/xapi/cmi5/extensions/launchmode https://w3id.org/xapi/cmi5
launchurl Fully qualified URL equivalent to the one the LMS used to launch the AU and without name/value pairs. Used as a context extension. https://w3id.org/xapi/cmi5/context/extensions/launchurl https://w3id.org/xapi/cmi5
publisherid A unique IRI for a single AU that is used to identify it as belonging to the publisher. Provided in Course Structure and must be in all cmi5 defined and cmi5 allowed Statements. Used as a context extension. https://w3id.org/xapi/cmi5/context/extensions/publisherid https://w3id.org/xapi/cmi5
moveon Controlled vocabulary which tells the conditions under which the AU is considered satisfied for the purposes of determining overall completion or prerequisites for other activities. Used as a context extension. https://w3id.org/xapi/cmi5/extensions/moveon https://w3id.org/xapi/cmi5
launchParameters The launch parameters used to launch this AU. Parameters are endpoint, fetch, actor, registration, and activityId. Used as a context extension. https://w3id.org/xapi/cmi5/extensions/launchParameters https://w3id.org/xapi/cmi5
