Dynamically Populate xAPI Verbs In A Dropdown List

This is an example of using JQuery to dynamically display a list of xAPI Verbs from the ADL Vocbaulary server (http://xapi.vocab.pub). The terms are provided as JSON data accessible through a SPARQL endpoint. In this example, we loop through results and display the terms in a drop down / select input form. These examples might be especially useful to content authoring tool developers or other xAPI tool developers.
